# PDF Studio

## Modules

La liste des modules affichés dans le sélecteur **MODULE** est désormais fournie par l'API backend `GET /pdf-studio/modules`.
Cela garantit une source de vérité unique pour les libellés et l'ordre d'affichage.

Les clés de modules historiques restent supportées côté backend via une normalisation transparente, afin de préserver la rétro-compatibilité des configurations existantes.
