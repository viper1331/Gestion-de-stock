# Vision capteurs ARI

Préparer l’intégration capteurs sans verrouiller la techno : stocker source, device_id, captured_at, et payload_json. Les mesures détaillées sont stockées dans ari_measurements pour supporter des flux multiples (avant/pendant/après). La V1 reste 100% manuelle, la V2 pourra consommer du Bluetooth/CSV/API sans changement de schéma.
